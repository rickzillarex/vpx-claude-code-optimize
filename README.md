# vpx-claude-code-optimize
optimized vbs scripts for VPX tables running on the ALP4k, using the script prompt from https://github.com/Bla1ze/vpx-claude-code-optimize

Pretty sure these are repo tables, but there's a chance that as Bigus and JP have updated their tables, I didn't keep up to date.

Many of these tables already play really well on the 4kp, so you may not see improvements until multiball. 

I also haven't deeply tested most of these tables, just confirmed they open and appear to work correctly, so lmk if there are breaking issues.
